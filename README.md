# CAC平台
### 一、使用方法
* 1.运行zlits_cac.sql文件；
* 2.修改application.properties数据配置.

### 二、相关框架集成
* 1.使用MyBatis实现ORM，PageHelper实现分页功能；
* 2.使用druid实现数据库连接池；
* 3.实现基于角色的权限配置：用户管理、角色管理、菜单管理；
* 4.使用LayUI进行前端页面展示；
* 5.使用自定义标签privilege.tld实现按钮级别的权限控制；
* 6.集成第三方框架：jedis、kafka.
